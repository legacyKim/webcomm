# 환경변수 보안 가이드

## 즉시 조치사항
1. JWT_SECRET 변경
2. AWS 키 재생성
3. Resend API 키 재생성 
4. Gmail 앱 비밀번호 재생성

3. 개발용과 프로덕션용 키를 분리
4. 정기적으로 API 키 로테이션
